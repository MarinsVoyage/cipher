Cipher

Cipher is a lightweight, dependency free converter for text, hexadecimal, and binary using UTF-8, ASCII, and EBCDIC (IBM CP037).

Highlights

- Text to hexadecimal, hexadecimal to text, text to binary, and binary to text.
- UTF-8, ASCII, and EBCDIC (IBM CP037) support.
- Works under `file://` and GitHub Pages with no build step.

Run It

- GitHub Pages: https://marinsvoyage.github.io/cipher/
- Run locally: open `index.html` in a modern browser, or serve the folder with `python -m http.server 8000` and visit `http://localhost:8000/`.

Configure

Edit JSON files under `config/`.
Field details live in `CONFIG.md`.
